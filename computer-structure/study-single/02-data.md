## 데이터

### 이진법 표기
- 정보 단위
  - 비트(bit) : 0과 1을 나타내는 가장 작은 정보 단위
  - 바이트(byte) : 비트 8개를 묶은 단위 (1byte = 8bit)
  - KB, MB, GB, TB 가 있으며 1,000개씩 묶어 표현
  - KiB, MiB, GiB, TiB 가 있으며 1,024개씩 묶어 표현

- 2진법 : 0과 1로만 숫자를 표현하는 방법
- 이진수의 음수 표현 - 2의 보수(2's complement)
  - 모든 0과 1을 뒤집고 1을 더하기
  - $2^n$ 과 $-2^n$을 구분할 수 없다는 한계가 있음
- 16진법 : 15를 넘어가는 지점에서 자리 올림하여 수를 표현하는 방법
  - 2진수와의 전환이 유리함

### 문자 표기 방법
- 문자 집합 : 컴퓨터가 인식하고 표현할 수 있는 문자의 모음
  - 문자 인코딩 : 문자를 0과 1로 표현된 문자 코드로 변환하는 과정
  - 문자 디코딩 : 0과 1로 표현된 문자 코드를 문자로 변환하는 과정
- 아스키 코드 : 초창기 문자 집합 중 하나, 7비트로 표현(128개의 문자)
  - 나머지 1비트는 패리티 비트로 오류 검출로 사용
- EUC-KR : 한글을 2바이트 크기로 표현할 수 있는 완성형 인코딩 방식
- 유니코드(unicode) : 여러 나라의 문자들을 광범위하게 표현할 수 있는 통일된 문자 집합이며, UTF-8, UTF-16, UTF-32등의 인코딩 방식이 있다.
  - UTF-8은 가변 길이 인코딩 방식으로 첫 번째 바이트의 앞부분을 보고 총 몇 바이트인지 알 수 있습니다.
